Technology & System

Technology

In this project, We use Twitter API as the primary data source for the sport event analysis.The data is pulled in live from Twitter API and stored in Google Cloud Storage for further analysis. The live data is also continuously processed and updated as new tweets are posted. And we process and transform the data before storing it in GCP Storage.

System Design

Dataset

To collect the dataset, we connect to the Twitter API and define specific hashtags, dates, and time ranges for retrieving tweets. We then analyze the sentiment of each tweet using a sentiment analysis tool. The dataset is organized with each entry representing a unique tweet, containing information such as the hashtag, time, text, location (if available), favorite count, and sentiment score. This comprehensive dataset enables a deeper understanding of public opinions and engagement with various topics on Twitter, as well as the exploration of trends, patterns, and insights related to the chosen hashtags.
